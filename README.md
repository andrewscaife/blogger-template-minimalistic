# Anti-Social Feed

A minimal Blogger theme for simple writing, journalling, and digital gardening. Built for clarity, fast reading, and a calm, no-clutter experience.

## Why this exists

I wanted a Blogger theme that felt quieter, simpler, and more intentional than the standard options.

Most Blogger themes feel either too busy, too dated, or too bloggy. This one strips things back and focuses on what matters. Clean typography, a simple layout, readable post lists, and just enough personality to feel human.

It is built for people who want a lightweight space for notes, stories, thoughts, photos, journalling, or a digital garden.

## Features

- Minimal, readable layout
- Clean homepage with intro and post index feel
- Simple navigation
- Expanding search icon
- Label pages styled like tidy archive pages
- Search results page with clearer heading text
- Footer with theme credit and optional theme download link
- External link indicators
- Lightweight Blogger XML theme with no unnecessary clutter

## Best suited for

- Personal blogs
- Digital gardens
- Online notebooks
- Journals
- Quiet writing spaces
- Minimalist Blogger sites

## Installation

1. Download the XML theme file from this repository.
2. In Blogger, go to **Theme**.
3. Click the dropdown arrow next to **Customize**.
4. Choose **Restore**.
5. Upload the XML file.
6. Save the theme.

## Customisation

You can edit the theme in Blogger under:

**Theme > Edit HTML**

A few things you may want to personalise:

- Site title
- Intro text
- Navigation links
- Footer links
- Theme download link
- Colours
- Fonts
- External link domain rule, if needed

## Support

This started as a personal project and grew from there.

It took a fair bit of trial and error to get it working the way I wanted, so it’s shared as-is and without support.

If you’re comfortable editing Blogger themes, you’ll be fine.

## Notes

This theme was built around a real site and then cleaned up for sharing, so it is practical first rather than over-engineered.

Blogger’s native search still has its usual limitations. For example, partial-word searches are not as smart as modern site search tools.
